FAQ

Les questions que l’on me pose le plus souvent.

Si vous ne trouvez pas la réponse à votre question, écrivez-moi via le formulaire de contact. Je préfère clarifier les choses avant que vous ne chargiez vos crédits.

Un texte généré dans une langue Standard consomme 1 crédit. Un texte généré dans une langue Premium consomme 2 crédits. Les langues en version bêta consomment 1 crédit, avec un marquage clair dans l’interface.

Oui. Les deux modes sont disponibles en permanence. Vous pouvez utiliser le mode accompagné pour les messages sensibles, et le mode expert pour les textes plus routiniers. Les crédits sont partagés entre les deux.

Vous pouvez soumettre une demande de révision en joignant le contexte saisi, le texte généré et une courte explication de ce qui pose problème. Si l’analyse confirme que le rendu est insuffisant, les crédits correspondants sont recrédités.

Non. Les packs de crédits sont utilisables librement et les abonnements mensuels peuvent être arrêtés à tout moment avant la date de renouvellement.

Oui, selon la formule choisie. Les packs peuvent être utilisés par une ou plusieurs personnes, et les abonnements à partir de l’offre Équipe prévoient plusieurs utilisateurs avec un suivi commun des crédits.

Les textes que vous saisissez sont utilisés uniquement pour générer vos messages et pour améliorer le service de façon agrégée. Les données sensibles peuvent être anonymisées et des bonnes pratiques vous sont rappelées dans le produit.

L’analyse d’un texte reçu est comptée comme une génération classique : vous utilisez vos crédits exactement comme pour la rédaction.

À titre indicatif :

  • Un email ou un message court (jusqu’à ~800 mots) correspond en général à 1 crédit en langue Standard ou 2 crédits en langue Premium.
  • Un document plus long (contrat, rapport, compte-rendu…) peut nécessiter plusieurs générations. Vous pouvez découper le texte en blocs pour mieux maîtriser votre consommation.

Vous gardez ainsi une vision claire du coût : les mêmes packs de crédits couvrent la rédaction, la traduction contextuelle et la compréhension de vos messages.
